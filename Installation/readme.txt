SystemPay Install
-----------------

1. Install NBrightSystemPay_*.*.*_Install.zip package as a normal DNN module.
2. Go into NBSv3 back Office.
3. Goto Admin>Plugins
4. Check the "SystemPay Payment" entry exists.
5. Go into "Admin>SystemPsy Panyment" option in the BO menu and enter the details required. ***IMPORTANT***
